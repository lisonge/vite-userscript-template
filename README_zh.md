# vite-userscript-template

一个简单的使用 vite-plugin-monkey 的示例

## 用法

```shell
git clone https://github.com/lisonge/vite-userscript-template.git
cd vite-userscript-template
pnpm install
pnpm run dev
# 在新打开的标签页里安装脚本
```

然后打开 <https://www.google.com/> 即可看到效果
